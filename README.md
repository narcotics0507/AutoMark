# AutoMark

一款支持 Chromium 浏览器的扩展（Chrome, Edge, Brave 等），利用 AI 模型（OpenAI GPT-4, Google Gemini, DeepSeek 等）智能分析、分类并清理您的浏览器书签。

## ✨ 核心功能

> [!IMPORTANT]
> **使用前建议**：虽然我们尽力确保安全，作为良好的习惯，建议您在进行大规模整理前**导出备份**您的书签 (Chrome 菜单 -> 书签 -> 书签管理器 -> 导出书签)。

### 1. 自动归类 (Smart Add)
添加书签时，AI 会自动分析标题和 URL，将其归类到最合适的文件夹。
*   **智能推荐**：自动移动到现有文件夹或创建新分类。
*   **非侵入式**：默认延迟 4 秒执行，为您预留手动操作时间；若期间手动移动书签，AI 自动取消。
*   **完全掌控**：提供右上角弹窗反馈，支持一键撤销或修改。

### 2. 批量整理 (Batch Organize)
支持对“书签栏”或“其他书签”进行全量扫描与重新归类。
*   **可视化预览**：执行前生成完整的移动/重命名计划，支持逐项确认。
*   **安全可靠**：您清楚知道每一个变动，避免误操作。

### 3. 维护工具
*   **失效链接检测**：快速扫描无法访问的死链，一键清理。
*   **重复书签清理**：识别并合并重复保存的链接，保持书签库整洁。

### 4. 配置与隐私
*   **多模型支持**：内置支持 OpenAI, Google Gemini, DeepSeek，兼容 One API 等自定义接口。
*   **隐私保护**：API Key 与配置仅存储在本地 (LocalStorage)。书签数据仅在分析时发送至您指定的 API 端点，绝不上传至第三方服务器。

## 🚀 安装与使用

1.  **加载扩展**
    *   打开浏览器的扩展管理页（`chrome://extensions/` 或 `edge://extensions/`）。
    *   开启 **开发者模式**。
    *   点击 **加载已解压的扩展程序**，选择本项目根目录。

2.  **配置**
    *   点击扩展图标进入设置。
    *   选择 API 提供商并填入 API Key。
    *   勾选 **“自动整理新添加的书签”** 开启智能归类功能。

3.  **使用**
    *   **日常**：像往常一样添加书签，AI 会在后台自动处理。
    *   **整理**：点击扩展图标 -> “开始智能整理”，按向导操作即可。
