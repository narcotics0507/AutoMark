# AutoMark

**简体中文** | [English](README_EN.md)


AutoMark 是一款支持 Chromium 浏览器（Chrome, Edge, Brave 等）的扩展。它利用大语言模型（OpenAI GPT-4, Google Gemini, DeepSeek 等）来理解您的书签内容，帮助您自动分类、重命名和清理，让书签栏保持井井有条。

## ✨ 核心功能

> [!IMPORTANT]
> **使用前建议**：虽然扩展已包含多重安全机制，作为良好的数据习惯，建议您在进行大规模整理前**导出备份**您的书签 (Chrome 菜单 -> 书签 -> 书签管理器 -> 导出书签)。

### 1. 智能归类与重命名 (Smart Add & Rename)
当您添加书签时，扩展会自动分析页面标题和 URL。
*   **自动归类**：将书签移动到最合适的现有文件夹，或根据内容创建新分类。
*   **智能重命名**：如果原标题含糊不清（如 "Home - Official Site"），AI 会建议一个更直观、简洁的名称。
*   **非侵入式体验**：操作后会弹出提示，默认延迟执行。您可以随时点击“撤销”，或者在弹窗中直接修改目标文件夹和名称。

### 2. 批量整理 (Batch Organize)
支持对某个文件夹（如“书签栏”）进行全量扫描与重构。
*   **可视化预览**：在真正执行任何操作前，系统会生成一份详细的“整理计划”。
*   **完全透明**：您可以看到 AI 建议创建哪些文件夹、移动哪些书签。不满意的地方可以随时取消勾选。

### 3. 安全备份 (Safe Backup)
我们极其重视您的数据安全：
*   **一键备份**：在控制台首页，您可以随时点击“备份书签”按钮，将当前所有书签导出为通用的 HTML 文件。
*   **自动备份**：在执行智能整理前，系统默认会自动备份您的书签。即使整理结果不满意，也能随时无损还原。

### 4. 书签库维护工具
我们提供了一套工具来保持书签库的健康：
*   **失效链接归档**：扫描无法访问的死链。为了防止误删（如临时网络问题），系统会将它们统一移动到 `失效链接归档` 文件夹，而不是直接永久删除。
*   **重复书签清理**：智能识别重复保存的链接。保留最早或位置最好的版本，清理冗余项。

### 5. 隐私与配置
*   **多模型支持**：内置支持 OpenAI, Google Gemini, DeepSeek，兼容 One API 等自定义接口。
*   **隐私保护**：API Key 与配置仅存储在本地 (LocalStorage)。书签数据仅在分析时发送至您指定的 API 端点，绝不上传至第三方服务器。

## 🚀 安装与使用

1.  **加载扩展**
    *   打开浏览器的扩展管理页（`chrome://extensions/` 或 `edge://extensions/`）。
    *   开启 **“开发者模式”**。
    *   点击 **“加载已解压的扩展程序”**，选择本项目根目录。

2.  **配置**
    *   点击扩展图标进入设置页面。
    *   选择 API 提供商并填入 API Key。
    *   您可以勾选 **“自动整理新添加的书签”** 来开启实时整理功能。

3.  **使用**
    *   **日常**：像往常一样添加书签，留意右上角的整理提示即可。
    *   **整理**：点击扩展图标 -> “开始智能整理”，跟随向导完成全库扫描。

---

## 🛠️ 排查问题
*   **操作反馈**：在整理过程中，控制台会实时显示当前的操作（如归档了多少链接）。
*   **详细日志**：如果遇到未知错误，请查看设置页面左下角的 **“系统日志”** 区域，这里记录了详细的 API 通信和网络状态，便于排查问题。
