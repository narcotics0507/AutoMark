# AutoMark (Chromium 扩展)

这是一款强大的 Chromium 浏览器扩展（支持 Chrome, Edge, Brave 等），利用先进的 AI 模型（OpenAI GPT-4, Google Gemini, DeepSeek 等）智能分析、分类并清理您的浏览器书签，助您轻松维护整洁有序的知识库。

## ✨ 核心功能

### 1. 🤖 自动分类 (Automatic Categorization)
当您添加新书签时，扩展会自动接管并进行智能处理（支持 Chrome 和 Edge）：
*   **自动分类**：AI 实时分析标题和 URL，将其移动到最合适的现有文件夹，或创建合理的层级结构。
*   **即时反馈**：右上角弹窗展示分类结果和 AI 的推荐理由。
*   **完全掌控**：
    *   **✅ 确认**：认可分类，立即关闭弹窗。
    *   **↩️ 撤销**：一键还原到初始位置。
    *   **✏️ 修改**：手动选择或创建其他文件夹。
*   **异常提醒**：如果 AI 服务中断或网络错误，会显示醒目的错误提示，确保您知情。

### 2. 📂 批量整理 (Batch Organize)
不仅仅是新书签，您的旧书签也能焕然一新：
*   **向导模式**：通过简单的步骤引导您选择需要整理的文件夹（如“书签栏”或“其他书签”）。
*   **可视化审查**：在执行任何更改前，生成详细的“整理计划”。您可以预览每一个移动、重命名或新建操作，并随时取消个别项目。

### 3. 🧹 维护工具箱
*   **💀 失效链接检测**：扫描并标记无法访问的书签，支持一键清理，保持书签库健康。
*   **👯 重复书签清理**：快速识别重复保存的链接，提供详细对比，由您决定保留哪一个。

### 4. ⚙️ 灵活配置与隐私
*   **多模型支持**：内置支持 OpenAI (GPT-3.5/4), Google Gemini, DeepSeek，同时支持自定义 OpenAI 兼容接口（如 One API）。
*   **本地日志**：内置“系统日志”查看器，**实时监控**扩展运行状态，方便排查 API 连接或分类失败的原因。
*   **隐私优先**：您的 API Key 和配置仅存储在本地浏览器（LocalStorage）。书签数据仅在您使用 AI 功能时发送至您指定的 API 端点，绝不上传至任何第三方服务器。

## 🚀 安装与设置

1.  **加载扩展**
    *   **Chrome 用户**：打开 `chrome://extensions/`。
    *   **Edge 用户**：打开 `edge://extensions/`。
    *   开启右上角（或左下角）的 **开发者模式 (Developer Mode)**。
    *   点击 **加载已解压的扩展程序 (Load unpacked)**，选择本项目根目录。

2.  **配置 AI**
    *   点击扩展图标进入 **设置页面**。
    *   选择 API 提供商并填入 API Key。
    *   勾选 **“自动整理新添加的书签”** 以启用 Smart Add 功能。
    *   点击 **保存配置** 并 **测试连接**。

## 📖 使用指南

### 场景一：日常收藏
只需像往常一样点击 Chrome 的星星图标添加书签。
*   几秒钟后，右上角会弹出 AI 的整理结果。
*   如果满意，点击 **“确认”** 或等待自动关闭。
*   如果不满意，点击 **“修改”** 手动调整。

### 场景二：整理旧书签
1.  点击扩展图标打开主页。
2.  点击 **“开始智能整理”**。
3.  勾选要整理的文件夹。
4.  (可选) 点击 **“检测失效链接”** 或 **“检测重复书签”** 进行专项清理。
5.  点击 **“开始智能分析”**，等待 AI 生成计划。
6.  审查计划无误后，点击 **“确认执行”**。

## 🛠️ 排查问题
如果发现功能不工作：
1.  进入 **设置页面**。
2.  查看左下角的 **“系统日志”** 区域。
3.  点击 **“刷新日志”**，查看具体的错误信息（如 API 额度不足、网络超时等）。
